El login es el proceso mediante el cual un usuario se identifica con sus credenciales para acceder a una aplicación. Esto permite que el sistema reconozca quién está utilizando la plataforma. Algunas rutas deben estar protegidas porque contienen información privada o funciones que solo pueden usar usuarios autorizados. De esta manera se evita el acceso no permitido. No deben exponerse públicamente contraseñas, documentos de identidad, teléfonos, direcciones, tokens, claves de API ni otros datos sensibles de la comunidad.
